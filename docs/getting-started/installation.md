# Installation
Get familiar with the basic setup and overview of Vue-Istilo.

[[toc]]

## Install Vue-Istilo via NPM
After creating a project with vue-cli or custom made (usually Webpack).

```bash
// Using npm
npm install vue-istilo

// Using yarn
yard add vue-istilo
```

## Configure your project to use vue-istilo
When the installation success, We will import the vue-istilo and use it.

```js
import Vue from 'vue';
import VueIstilo from 'vue-istilo';

Vue.use(VueIstilo)
```
